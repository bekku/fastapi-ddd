## セットアップ手順

### 環境構築
1. このリポジトリをクローンします。
   ```bash
   git clone https://github.com/labdemy-dev/fastapi-ddd.git
   cd fastapi-ddd
   ```
2. 下記の.envを`fastapi-ddd/.env`に配置

5. イメージのbuild
   ```bash
   docker compose build
   ```
6. 実行
   ```bash
   docker compose up
   ```
